# SenoraRP Chatlog Assistant
This program is used to convert the chat logs generated while playing on SenoraRP into readable text.


## Getting Started

No installation is required. Simply download the latest [release](https://github.com/SENRP/SenoraRP-Log-Parser/releases) and run the executable.
